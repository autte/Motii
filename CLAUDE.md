> 每次开始开发前请先阅读此文件。

## 项目简介

**Motii** 是一个面向脑瘫儿童家长的 PWA（渐进式网页应用），帮助家长：
1. 用手机视频 + AI 自动分析孩子的运动质量
2. 追踪康复进展（关节角度、步态、平衡等）
3. 计算不同运动的康复价值（ROI）
4. 每月记录关键指标，可视化进步趋势

**目标用户：** 在加拿大生活的脑瘫儿童家长，尤其华人家庭。
**未来方向：** 加拿大创业，申请 AI 医疗健康类政府基金。

## 重要医学注意事项（影响Motii设计）

1. **右侧偏盲风险**：MRI显示左额叶损伤，需定期眼科检查；视觉训练要覆盖右侧视野
2. **眼部追视在中线停止**：这是右眼外斜的根源，视频分析需特别关注眼球运动
3. **精细运动远好于粗大运动**（FMQ=73rd vs GMQ=4th）：无人机/电子制作ROI判断正确
4. **肘关节痉挛2/4**：右手举过头顶的动作需注意，不强迫
5. **疲劳加重代偿**：报告明确指出疲劳→摔跤频率增加，运动量需严格控制

- **前端：** 纯 HTML + CSS + JavaScript（单文件 PWA）

- **AI：** Anthropic Claude API（claude-sonnet-4-6）
- **存储：** 浏览器 localStorage（本地，不上传）
- **部署：** Vercel（主）+ GitHub Pages（备）

## 设计原则

1. **移动优先**：按手机屏幕设计（375px宽）
2. **5分钟原则**：每次记录不超过5分钟
3. **数据最小化**：视频不上传，只传截帧图片
4. **兴趣驱动**：兴趣活动纳入康复评估
5. **家长视角**：输出决策建议，不是医学术语
